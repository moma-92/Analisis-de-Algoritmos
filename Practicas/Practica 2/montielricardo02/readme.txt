El programa solo recibe Archivos con terminacion .txt

Para que el programa tenga exito se requiere la ruta completa del lugar donde se tiene almacenado el Archivo .txt
OJO: PUNTO IMPORTANTE: La ruta proporcionada no puede tener espacios.

Una forma de obtener la ruta de donde se tiene almacenado el archivo es abriendo la terminal en el mismo lugar donde
se tiene el archivo e insertamos el comando " pwd " en caso de Linux.

Haciendo esto podemos obtener la ruta donde se encuentra el archivo:

EJ: /home/richar/Escritorio

Ahora a esto le falta que le agreguemos el nombre del Archivo .txt

Entonces obtenemos:

/home/richar/Escritorio/Archivo.txt

Damos enter para que el programa se ejecute.

Dentro de la carpeta src viene un ejemplo del formato de que deben de seguir los Archivos, si es que se el quieren
pasar otros archivos. Este mismo archivo es ejecutable, solo hay que proporcionar la ruta.
